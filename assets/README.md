# Brand Assets

Official logos live in `assets/images/` (source of truth).

Web apps serve copies from:

- `apps/web/member-app/public/brand/`
- `apps/web/admin-portal/public/brand/`

| File | Usage |
|------|--------|
| `welfare-logo.jpg` | Member portal, auth screens |
| `batch-logo.jpg` | Admin portal |

Brand colors are defined in `packages/config/src/brand.ts`.
